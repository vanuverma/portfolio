Check On Me

Emergency alert and wellness check platform

Mobile + Web Full Stack Real-time Notifications

Overview

Check On Me is an emergency wellness system that automates check-ins and notifies trusted contacts when a user does not respond. The solution combines mobile, backend, and admin tooling to provide reliable alerts and visibility.

Technology Stack

Mobile

  • Flutter
  • Riverpod
  • OAuth Authentication

Backend

  • Node.js
  • TypeScript
  • PostgreSQL + Redis

Dashboard

  • React
  • Vite
  • TanStack Query

Infrastructure

  • Docker
  • Bull Queues
  • Twilio + SendGrid

Key Features

📅

Scheduled Check-ins

Configurable wellness schedules with reminders and missed-check escalation.

👥

Emergency Contacts

Priority-based contact lists with multiple notification channels.

📍

Location Sharing

Optional location context included in critical alerts.

🔄

Resilient Delivery

Retry-capable queue processing for dependable message delivery.

Technical Highlights

Background Processing

Implemented hybrid scheduling across mobile and backend services to keep alerts reliable even under intermittent connectivity.

Cross-Platform Support

Optimized dependency choices to support a broad range of iOS and Android versions while keeping security and authentication robust.

Developer Workflow

Automated local startup for all services to reduce setup friction and improve delivery speed.

Impact

  • End-to-end working product across mobile app, API, and admin dashboard
  • Reliable multi-channel notifications with retry and priority controls
  • Improved response readiness for users and emergency contacts
