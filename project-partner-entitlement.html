Service NSW Partner Integration QR Code System

Overview

A partner entitlement verification system that enables venues (such as RSL clubs) to verify customer entitlements through QR code scanning. The platform allows venue partners to self-onboard, generate unique QR codes, and verify customer credentials with cryptographic security and user consent management.

System Architecture

Partner Setup Flow

  1. Partner Registration: RSL club or venue user accesses Service NSW Partner Portal website
  2. Authentication: Partner authenticates via Auth0 identity provider
  3. Validation: System validates partner credentials and authorization
  4. Self-Onboarding: Partner completes onboarding process on Digital License Platform
  5. QR Code Generation: System generates unique QR code for the venue
  6. Configuration Storage: Venue-specific config is stored in MongoDB database
  7. QR Code Display: Partner portal displays QR code to visitors at venue

Customer Verification Flow

  1. QR Scan: Customer scans venue's unique QR code using Service NSW Mobile App
  2. Config Retrieval: System retrieves venue-specific configuration from MongoDB via PROXY
  3. User Consent: Customer is prompted to consent to share licence data
  4. Data Transmission: Upon consent, licence data is sent to venue system
  5. Cryptographic Validation: PKI vendor performs cryptographic verification
  6. Verification Result: Kubernetes platform processes and returns verification result to partner
